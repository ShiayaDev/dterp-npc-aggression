A lightweight FiveM script that makes NPCs passive, unarmed, and non-aggressive.
<img width="900" height="27" alt="image" src="https://github.com/user-attachments/assets/e889ed68-d88e-4ed8-95d0-3a0996da74eb" />



## Features
✔ Makes NPCs passive  
✔ Removes all weapons from NPCs  
✔ Disables aggressive combat behavior  
✔ Reduces NPC awareness (hearing & sight)  
✔ Prevents NPC reactions to events  
✔ Optimized (runs once per NPC)  

## Installation

1. Download the latest release
2. Place the folder in your `resources` directory
3. Add to your `server.cfg`

ensure dterp-npc-aggression

## Requirements

None (Standalone)
